# Pie Container

# Install

```
npm install # install the dependencies
npm link # link pie-container as a global executable

```

### Run a sample

```
cd test/resources/sample-content/bower-style/sample-item
pie-container 
```

### Tests

```
grunt test

```
##### debug tests: 

```
npm install -g node-inspector
node-debug _mocha test/spec/**
```
